Proyectos

login Validator

Al desarrollar el login validator utilizando Python y Flask se gestiona la autenticación de usuarios. La aplicación presenta formularios HTML y estilos CSS, proporcionando una interfaz atractiva. Valida las credenciales ingresadas, ofreciendo retroalimentación instantánea sobre el acceso, asegurando una experiencia de usuario segura y fluida.

Gestión de usuarios con CRUD

Implementación de una aplicación CRUD utilizando Flask y MySQL para gestionar datos de manera eficiente. La interfaz está construida con HTML y CSS, ofreciendo una experiencia de usuario intuitiva. Esta aplicación permite crear, leer, actualizar y eliminar registros, garantizando la integridad y seguridad de la información en todo momento.

Administrador de blog

Desarrollé un blog de noticias utilizando Flask y MySQL, con una interfaz intuitiva diseñada en HTML y CSS. La aplicación permite a los usuarios publicar, editar y eliminar artículos, así como comentar en las publicaciones. Implementé funciones de autenticación para usuarios, garantizando una experiencia segura y personalizada.

Data analytics

Realicé un proyecto de análisis de datos en energía solar utilizando Python. Apliqué técnicas de análisis y visualización para evaluar el rendimiento de sistemas fotovoltaicos, identificando patrones en la producción de energía y optimizando la toma de decisiones. Este enfoque me permitió ofrecer recomendaciones basadas en datos para mejorar la eficiencia energética
